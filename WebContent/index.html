This image provides the following services:
  • SAP OpenUI5 library
  • Get a list of all Hana procedures
  • Invoke any Hana procedure e.g. POST http://..../protected/rest/procedures/SYS/GET_OBJECT_DEFINITION with Json payload {"SCHEMA": "PUBLIC", "OBJECT": "VIEWS" }
  • Lookup a single (first) row in any table or view
  • Access any table or view via oData
  • Get the swagger definition of all APIs
  • Mount point where any custom UI5 application can be added
  • Mount point where any non-protected content can be added, e.g. login pages
